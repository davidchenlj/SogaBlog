本博客采用一下工具

1.后端采用的是django1.8

2.后台采用的是green网上找的简易后台模板

3.博客首页是 Bootstrap 3x

4.评论系统自己也弄了个,后面还是用 "多说" 评论系统.

5.编辑器采用Simditor v2.3.6

6.前端代码格式采用highlight.js v9.10.0
